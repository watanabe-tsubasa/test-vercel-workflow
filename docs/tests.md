- [x] 箇条書きを入力し「日記を生成」を押すと workflow が開始され、ステータスが「生成中」に変わる

- [ ] 文章を編集して「修正を送信」するとステータスが「画像生成中」になり、完了後に画像が表示される

- [x] フロー完了後、`/` の一覧に新しい日記が表示され、`/diary/[id]` で内容と画像が見える
